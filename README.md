## Setup

1. Clone the repository:
```bash
git clone [repository-url]
```

2. Open `index.html` in your browser or use a local development server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx serve
```

3. Visit `http://localhost:8000` in your browser
